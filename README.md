# Lab3-OOP

## Description
This project implements a matrix and vector system in C# using Object-Oriented Programming principles such as encapsulation, composition, aggregation, and dependency.

## Features
- Vector addition, subtraction, and scalar multiplication
- Matrix addition, multiplication, and determinant calculation
- Input validation and error handling

## How to Run
1. Open `Lab3-OOP.sln` in Visual Studio.
2. Build the solution.
3. Run the program and follow console instructions.
